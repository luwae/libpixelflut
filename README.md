## Features
- support for all pixelflut commands (except `HELP`)
  - size: `SIZE`
  - get pixel: `PX <x> <y>`
  - put pixel: `PX <x> <y> <rrggbb(aa)>`
- useful error messages
- optional write and read buffering, and batched use of commands
